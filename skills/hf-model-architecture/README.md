# HF Model Architecture

Capture Hugging Face model architecture diagrams via [hfviewer](https://hfviewer.com/).

**npm:** [`hf-model-architecture-skill`](https://www.npmjs.com/package/hf-model-architecture-skill)

**Full documentation:**

- [English (repository root)](../../README.md)
- [简体中文 (仓库根目录)](../../README.zh-CN.md)

**Quick start** (from npm):

```sh
npx hf-model-architecture-skill zai-org/GLM-5.2 --out artifacts/glm-5.2 --level 4
npx playwright install chromium
```

**From repository root:**

```sh
npm install
npx playwright install chromium
npm run capture -- zai-org/GLM-5.2 --out artifacts/glm-5.2 --level 4
```

**Agent instructions:** [SKILL.md](./SKILL.md)
